HTML Cheat Sheet

Basics

<html> Goes around the whole document.
<body> contains all visible elements
<h1-h6> Headlines in order, the main headline is a h1 and then going down to h6 or however many necessary
<p> for all text that is not a headline
<br> creates a line break. Note: It is better practice to use padding or margins instead.
<div> Specifies a logical division within a document. Use it to separate or identify chunks of content that are not otherwise distinguished naturally using other tags. One of the most common HTML tags.
<span> An inline span within text that goes with the flow of content (whereas a <div> tag is block-level and breaks the flow)
<a> Anchor (most commonly a link)
<link> Defines a relationship to another documente, this can be page which is part of the site, an anchor or an external document or site.
<img> Show an image

Table

<table> Use for repeated data that has a naturally tabular form. Never use for layout purposes.
<caption> Caption for a table: describes the tableâ€™s contents
<th> Table column or row header cell. May appear in a <thead> (to denote a column header cell), <tbody> (to denote a row header), and in <tfoot> (to denote a column foot cell, e.g. a total)
<tr> Table row
<td> defines a standard cell in an HTML table.
<thead> headline in a table
<tbody> Indicates the main body of a data table
<tfoot> Indicates the footer of a data table
<col> Identifies a particular column in a table

Frames

<iframes> An inline frame is used to embed another document within the current HTML document.
<frameset> The <frameset> tag defines a frameset. The <frameset> element holds one or more <frame> elements. Each <frame> element can hold a separate document. (Not supported in html 5!)
<noframes> The <noframes> tag is a fallback tag for browsers that do not support frames. It can contain all the HTML elements that you can find inside the <body> element of a normal HTML page.

Lists

<ul> Unordered list
<ol> ordered list
<li> list item (in both li and ol lists)
<dl> Contains one or more definition-term / definition-description pairs.
<dt> Represents the legend of a figure
<menu> Menu item list. Deprecated. Do not use. Use other standard list types instead.
