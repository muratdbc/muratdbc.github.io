Ruby Enumerable Map

08/31/2014

This post will dive into one the most used built-in method that you get when you include Ruby's most useful Module :Enumerable. What is Module? Module contains bunch of methods packaged together and get added (proper term is mixed-in) to classes such as Hash and Arrays.

Enumerables module comes with really useful method.like #map,so my main goal is to get well acquinted with it. In Ruby doc definition, this is what map does: Returns a new array with the results of running block once for every element in enum.

Map in essence iterates over the elements of the array and every element will be applied what is defined in the block part. The block part can be any code kind of code. Of course If you use only #map method the actual array will not be changed. In order to change the actual array, you have to use destructive one which is #map!
