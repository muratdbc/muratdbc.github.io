Margin,Padding and Border

08/15/2014

Today I will talk about margins, padding and borders. But before we dive into I would like to mention about the box model.ok but lets start with basics. HTML is a markup language for describing the strcuture of web pages. HTML is a tree-like structure where each node can be called as element. Cascade Style Sheets(CSS) contols the presentation of the the web page. So to speak, CSS controls how HTML element is represented.CSS treates every element as if it were represented by a box. Below are the images of a box model.

As you can see every element has a content, which could be a text or image and this content is placed inside a box that is just big enough to contain it. Notice that content area has not white space.Padding is used to create visual space whitespace between the content and the border of the box. Padding is transparent and has no color or decoration of its own. Border surrounds the padding and, becasue it takes the form of a line around your content,provides visual seperation between your content and other elements oon page.Margin gives a way to add space between your element and other spaces.
